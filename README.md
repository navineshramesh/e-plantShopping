repo name: e-plant shopping application

project name: Paradise nursey application

Overview: Paradise Nursery is a responsive e-commerce web application designed for browsing and purchasing a diverse selection of indoor and outdoor plants. The platform categorizes products by functional and aesthetic traits, such as air-purifying, low-maintenance, and decorative varieties, to help users find plants suited to their environment. Built with a mobile-friendly layout and an interactive interface, the application streamlines the digital shopping experience from initial catalog exploration to final checkout management.
